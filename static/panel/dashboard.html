Infraestructura

Listo para producción.

Construido con FastAPI, SQLAlchemy y autenticación básica. Desplegable en Railway o Render con una variable de entorno.

FastAPI + Uvicorn

API asíncrona de alto rendimiento con documentación OpenAPI autogenerada en /docs y /redoc.

SQLite / PostgreSQL

SQLite para desarrollo local. PostgreSQL en producción — solo cambia DATABASE_URL en el .env.

Gestión de Usuarios

Registro, autenticación por token y passwords hasheados con bcrypt. Token con expiración configurable.

Rate Limiting

Slowapi integrado para proteger los endpoints públicos contra abuso y scrapers.

Stats en Tiempo Real

Endpoint /api/stats con conteo de tareas por estado, categorías y usuarios activos.

Deploy-ready

Incluye render.yaml y Procfile para despliegue inmediato en Render o Railway sin configuración extra.
