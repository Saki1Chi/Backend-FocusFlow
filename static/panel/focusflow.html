Funcionalidades

Todo lo que necesitas, en tu mano.

Un conjunto de herramientas diseñadas para que nunca pierdas el hilo de tu día.

Vista Calendario

Visualiza tus tareas en un calendario mensual / semanal con indicadores de estado y prioridad.

Tareas Recurrentes

Configura reglas de recurrencia: diaria, semanal, mensual. Nunca vuelvas a crear la misma tarea dos veces.

Bloqueo de Apps

Bloquea aplicaciones distractoras en Android mientras tienes tareas pendientes. Foco real.

Alarmas Inteligentes

Recordatorios precisos con persistencia entre reinicios del dispositivo via Android Alarm Manager.

Sincronización Cloud

Tus tareas se sincronizan con el backend FocusFlow CMS. Accede desde cualquier lugar.

Colores Personalizados

Color picker integrado para personalizar el acento visual de cada categoría de tareas.
