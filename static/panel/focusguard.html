FocusGuard

El guardián de tu productividad. Bloquea juegos y distracciones automáticamente hasta que termines tus tareas — con privilegios de administrador y arranque automático.

Descargar .exe v1.0.0 · Windows 10/11 · Requiere Admin

Se registra en el Task Scheduler para arrancar con el sistema

Funcionalidades

Sin distracciones. Sin negociación.

FocusGuard corre como daemon en segundo plano y se asegura de que solo abras juegos cuando lo merezcas.

Bloqueo de Juegos

Lista negra configurable de ejecutables (.exe). League of Legends, Valorant, Fortnite — lo que sea que te distraiga.

Focus Blocks

Sistema de bloques de trabajo tipo Pomodoro. Completa N bloques para desbloquear un intervalo de juego configurado.

Tareas del Día

Si completas todas tus tareas, el bloqueo se levanta automáticamente. La recompensa se gana, no se pide.

Hora de Desbloqueo

Configura una hora límite (p.ej. 21:00) a partir de la cual el bloqueo se levanta sin importar el progreso.

System Tray

Icono discreto en la bandeja del sistema. Control total sin ocupar espacio en la barra de tareas.

Historial de Productividad

Racha de días consecutivos con todas las tareas completadas. Datos guardados localmente en JSON.
